## Description

A web application that initiates a Server-Side Event communication with a simple Node.js server. The server sends new data to the frontend every 5 seconds.

## Getting Started

### Prerequisites

It is necessary:
- Docker
- Docker Compose

### Installation

Run `docker-compose up --build`

### Usage

Open `localhost:3000` in a web browser.
